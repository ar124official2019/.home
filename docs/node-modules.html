Node Modules

Node Modules feature will install some sensible global node modules. These modules will be installed for the installed node and npm.

Installed Modules are:

ModuleInfo
nodemonMonitor for any changes in your node.js application and automatically restart the server - perfect for development
pm2PM2 is a production process manager for Node.js applications with a built-in load balancer. It allows you to keep applications alive forever, to reload them without downtime and to facilitate common system admin tasks.
serveserve helps you serve a static site, single page application or just a static file (no matter if on your device or on the local network). It also provides a neat interface for listing the directory's contents:
@angular/cliAngular CLI - The CLI tool for Angular. Read more about Angular.
@ionic/cliThe Ionic command line interface (CLI) is your go-to tool for developing Ionic apps.. Read more about Ionic.
react-scriptsThis package includes scripts and configuration used by Create React App.. Read more about React.js and Create React App.
sequelize-cliThe Sequelize Command Line Interface (CLI). Read more about Sequelize and Sequelize CLI Tool.
