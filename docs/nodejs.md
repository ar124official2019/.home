## NodeJS
NodeJS recipe installs and configures following components.

- `fnm`: Fast Node Version Manager <a class="nf em" href="https://bit.ly/4au4RC5"></a>
- Latest NodeJS (LTS)
- Some Global NodeJS Modules

### Required Dependencies
- `curl`
- `unzip`
