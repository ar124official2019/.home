Terminal

Terminal recipe installs and configures following components.

  • Patched Nerd Fonts
  • Alacritty Config
  • Oh My Zsh
  • Tpm (Tmux Package Manager)
  • nvChad (Config for nvim - Neovim)

Patched Nerd Fonts

It will install some patched Nerd Fonts to user fonts directory ~/.fonts. See exuasted list of fonts installed here. These fonts are used by terminal, as per current config.

Alaritty Config

It will install included alacritty.yml to user home (~/.alacritty.yml). Following are high level settings for Alacritty.

SettingValueInfo
ThemeAlaritty NordSee Alacritty Nord.
FontFira Code Nerd FontSee Nerd Fonts.
TERMxterm-256colorSets the env.TERM to xterm-256color.

Oh My Zsh + .zshrc

It will install Oh My Zsh, a framework for ZSH. It also copies included .zshrc to user home (~/.zshrc). See following tables for variables, aliases and plugins.

  • Variables
  • Aliases
  • Plugins

TPM - Tmux Plugin Manager + .tmux.conf

It install TPM to user home directory at ~/.tmux/plugins/tpm. It also copies included .tmux.conf to user home directory as ~/.tmux.conf. TPM is enabled and configured in ~/.tmux.conf. See following tables for more info.

  • Tmux Plugins
  • Tmux Enabled Features
  • Tmux Keybindings

nvChad

It will install nvChad - a Blazing fast Neovim config providing solid defaults and a beautiful UI. It will automatically install default configured plugins and enable settings on your next start to nvim. It should install it to `~/.local/config/nvim'.

You should run nvim right after installing terminal feature and let it config automatically.

Nothing is touched on newly installed nvChad so it is nothing else except a try to install nvChad. Yet I've documented some key default keybindings here.
